# PHP 生成靜態部落格

## PHP Requirement
* PHP >= 7.4

## 功能
* 支援Markdown寫文章、頁面
* 搜尋
* 分頁

## Install packages:
```
$ npm install
```

## Watch js
```
$ npm run watch
```

## Build js
```
$ npm run build
```

## Start dev server
```
$ php serve
```

Serving at http://localhost:8000

## Build static page
```
$ php build
```
